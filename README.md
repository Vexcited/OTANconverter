# [OTANconverter](https://otan-converter.vexcited.com/) [![wakatime](https://wakatime.com/badge/user/0839e595-e07a-435c-8d59-ed95f2a3d6dd/project/3f90f3d4-b32e-47b6-90ff-e3c6e07f7a26.svg)](https://wakatime.com/badge/user/0839e595-e07a-435c-8d59-ed95f2a3d6dd/project/3f90f3d4-b32e-47b6-90ff-e3c6e07f7a26)

A simpler converter from text to OTAN, made for a Worldskills training challenge.

## Development

```bash
pnpm install
pnpm dev
```

Open your browser and navigate to `http://localhost:5173/` to see the app in action.
