# [OTANconverter](https://otan-converter.vexcited.com/)

A simpler converter from text to OTAN, made for a Worldskills challenge.

## Development

```bash
pnpm install
pnpm dev
```

Open your browser and navigate to `http://localhost:5173/` to see the app in action.
